# martip_mobile

A new Flutter project.

## Instalasi

### Prasyarat

Pastikan kamu sudah menginstall:

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (versi ≥ 3.12.1)
- [Android Studio](https://developer.android.com/studio) atau [VS Code](https://code.visualstudio.com/) dengan Flutter extension
- Git

### Langkah Instalasi

1. **Clone repository**
   ```bash
   git clone https://github.com/Zann-Aidil/MartipApp_Mobile.git
   ```

2. **Masuk ke direktori project**
   ```bash
   cd MartipApp_Mobile
   ```

3. **Install dependencies**
   ```bash
   flutter pub get
   ```

4. **Jalankan aplikasi**
   ```bash
   flutter run
   ```

> **Catatan:** Pastikan device/emulator sudah terhubung sebelum menjalankan `flutter run`. Cek dengan perintah `flutter devices`.
